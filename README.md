# cj-components
Basic components to generate sales funnels and touchpoints from the CustomerJourney.js project in javascript with a delicious vanilla flavor

## Components
- CardsList
- CtaBanner
- CountdownTimer
- HeroBanner
- imageBanner
- ImageText
- LevelCentered
- MediaList
- ModalBox
- OmnichannelChat
- StepsComponent
- TextAccordion
- TextColumns
- TextMessage
- TextNotification
- TextSection
- VideoPlayer
- WebinarInvitation
